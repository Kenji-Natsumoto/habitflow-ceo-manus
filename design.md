# HabitFlow for CEO - デザイン設計書

## コンセプト

経営者が事業成功に必要な習慣を毎日トラッキングするためのアプリ。自己啓発と実務の両面から、CEOとしての成長を支援する。

## ターゲットユーザー

起業家、経営者、事業責任者など、ビジネスの成功を目指すリーダー層

## デザイン方針

ダークモードをデフォルトとし、プロフェッショナルで洗練された印象を与える。モバイルポートレート（9:16）での片手操作を前提とした設計。

## スクリーン一覧

### 1. ホーム画面（Today）
メインのダッシュボード。今日の習慣チェックリストを表示。

**コンテンツ:**
- 日付表示（今日の日付）
- 進捗サマリー（完了数/全体数、達成率のリング）
- 習慣カテゴリ別セクション
  - 自己啓発（Mind）: 読書、瞑想、運動、学習
  - 実務（Business）: 売上確認、チーム1on1、戦略思考時間、顧客対応
- 各習慣はタップでチェック/アンチェック
- 連続達成日数（ストリーク）表示

### 2. 統計画面（Stats）
週間・月間の達成状況を可視化。

**コンテンツ:**
- 週間カレンダーヒートマップ
- カテゴリ別達成率グラフ
- 最長ストリーク記録
- 今週の達成率サマリー

### 3. 設定画面（Settings）
習慣のカスタマイズとアプリ設定。

**コンテンツ:**
- 習慣の追加・編集・削除
- リマインダー設定
- テーマ切り替え（ダーク/ライト）
- プレミアム機能（将来実装）

## 主要ユーザーフロー

### 習慣チェックフロー
1. アプリ起動 → ホーム画面表示
2. 今日の習慣リストを確認
3. 完了した習慣をタップ → チェックマーク付与 + ハプティクス
4. 進捗リングがリアルタイム更新
5. 全完了時 → 祝福アニメーション

### 統計確認フロー
1. タブバーから「Stats」をタップ
2. 週間ヒートマップで達成状況を確認
3. カテゴリ別の達成率を確認
4. モチベーション向上

## インタラクション設計

### 習慣チェック
- タップ時: Scale 0.97 + Haptics Light
- チェック完了: Success色のチェックマーク + Haptics Success
- 全完了: 祝福バイブレーション + 視覚的フィードバック

### リスト操作
- スクロール: スムーズなネイティブスクロール
- プルダウン: 日付リフレッシュ（将来実装）

## データ永続化

AsyncStorageを使用したローカルストレージ。クラウド同期は将来のプレミアム機能として検討。
